##AI Internship Tracker
A clean, responsive dashboard built to track internship applications, interviews, and response statuses in one organized place.

#Tech Stack
React + Vite

Tailwind CSS

#Key Features
Dashboard Overview: At-a-glance metrics for total applications, pending responses, and upcoming interviews.

Status Tracking: Group and manage applications by stage (e.g., Applied, Interviewing, Offered, Rejected).

Application Management: Easily add or update company details, roles, deadlines, and application links.

Responsive Design: Optimized layout for both desktop and mobile screens.
